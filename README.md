# NE Broadcast Suite — Releases

This repository hosts **release binaries and the auto-update feed** for
**NE Broadcast Suite**, the esports broadcast production app by Armour Studios,
distributed by Nameless Esports.

- **Download:** https://studio.namelessesports.com
- **Latest installer:** see [Releases](../../releases/latest)
- Installed copies update themselves automatically from this repository.

The application source code is developed in a private repository.
