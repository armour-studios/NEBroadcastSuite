Valorant — Live Client Data API

      

        Connects the Valorant overlay (localhost:3000/valorant.html)
        to Valorant's built-in Local Client API running on port 2999.
        No config files or plugins required — the connection is fully automatic.
      

      

        Valorant: waiting
      

      

        Setup steps
        
    
          
  1. Launch Valorant and join or spectate a match.
    2. 
          
  2. In BroadcastStudio, set the active game to Valorant on the Dashboard.
    3. 
          
  3. The overlay connects automatically — no restart needed.
    4. 
        

        

          For observer/spectator coverage:

          Use Valorant's built-in Observer Tool (requires a custom game or tournament lobby with spectator slots enabled).
          The overlay automatically highlights the currently-spectated player with a team-colored glow.
          No third-party software needed.
        

        

          How it works:
          Valorant exposes a local HTTPS server at 127.0.0.1:2999 during any active match.
          BroadcastStudio polls /liveclientdata/allgamedata every 500 ms and broadcasts
          real-time player health, weapons, abilities, economy, round state, and spike events to the overlay via WebSocket.
